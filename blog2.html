Commit Messages in Github

As we use github very much its also important to make sure our commits have a good commit messages.Now lets see few commit messages and their use cases which are good to make as a habit.

  • feat - a new feature
  • fix - a bug fix
  • docs - changes in documentation
  • style - everything related to styling
  • refactor - code changes that neither fixes a bug not adds a feature
  • test - everything related to testing
  • chore - updating build tasks, package numbers etc.
